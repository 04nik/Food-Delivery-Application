# upskillcampus

This is a web-based Food Delivery Application developed using **Python** and **Django**. The project provides an online platform where users can browse food items, add them to cart, and place orders easily.

## Features
- User Registration and Login System
- Browse Food Items / Menu
- Add to Cart Functionality
- Order Placement System
- Admin Dashboard for managing food items and orders

## Tech Stack
- Backend: Python (Django Framework)
- Frontend: HTML, CSS
- Database: SQLite
- Version Control: Git & GitHub

## Project Structure
- Django project with apps for handling users, orders, and menu
- Templates for UI rendering
- Admin panel for backend management

## Objective
To build a simple and efficient online food ordering system using Django.

## Future Improvements
- Online Payment Integration
- Order Tracking System
- Responsive UI (Mobile Friendly)
